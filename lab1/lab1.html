The Black Saint and the Sinner Lady

The Black Saint and the Sinner Lady es uno de los más grandes logros de orquestación en la historia del jazz. Charles Mingus diseñó conscientemente el ballet de seis partes como su magnum opus que -- puesto en evidencia por la famosa inclusión de las notas de su psicólogo -- es tabién tanto una examinación de su propio psique torturado como una pieza conceptual acerca del amor y la lucha. Allmusic,Steve Huey

Blue Train

A pesar de que nunca fue formalmente firmado, Blue Train fue publicado en honor a un acuerdo oral entre John Coltrane y el fundador de Blue Note Records, Alfred Lion, volviéndose la única colección de Coltrane publicada bajo la firma. El disco está repleto de evidencia sólida de las habilidades innatas de liderazgo de Coltrane. Allmusic, Lindsay Planer.

URL

Uniform Resource Localizer, una cadena de caracteres que corresponde únicamente a la ubicación de un recurso en la red, que pemite que sea encontrado por un navegador.


MIME

El tipo de Multipurpose Internet Mail Extensions es un estándar para indiar el tipo y formato de un documento. Permite al navegador decidir cómo procesar los datos que recibe.


XML

Extensible Markup Language, permite al usuario definir conjuntos de etiquetas o "markups" para facilitar la lectura y procesamiento de conjuntos de datos.


XHTML

Añade a html etiquetas de otras herramientas xml y facilita el procesamiento de ese tipo de documentos.


HTML5

Última revisión del estándar HTML, incorpora XHTML y añade más componentes con significado semántico.


Javascript

Variación del estándar ECMAScript, un lenguaje de programación orientado a objetos, basado en prototipos, imperativo, débilmente tipado y dinámico que se utiliza para dinamizar y mejorar interfaces web desde el código que ejecuta el cliente.


php

Lenguaje de programación de código abierto orientado al desarrollo web que puede ser incrustado en HTML. El código escrito en este lenguaje se ejecuta en el servidor (por lo que permanece oculto del usuario), generando HTML y mandándoselo al cliente.


JSP

Java Server Pages permite la creación de páginas web dinámicas basadas en HTML y XML de la misma forma que PHP, pero utiliza Java como lenguaje de programación. Es necesario correrlo en servidores compatibles con contenedores servlet.


AJAX

Acrónimo de Asynchronous JavaScript and XML, premite la creación de aplicaciones interactivas o Rich Internet Applications que se ejecutan en el cliente mientras se mantiene una comunicación asíncrona con el servidor en segundo plano (sin intervenir con la visualización o el comportamiento de la página). Elimina la necesidad del actualizado de páginas, por lo que aumenta la velocidad y usabilidad de las aplicaciones.


Métodos de petición HTTP

GET

Solicita un recurso específico, solo debe recuperar datos.


HEAD

Únicamente pide el header de una respuesta.


POST

Envía cierto recurso a una entidad, causando cambios en el servidor.


PUT

Reemplaza todas las presentaciones de cierto recurso en el servidor por lo que se envía junto con la petición.


DELETE

Borra el recurso especificado del servidor.


PATCH

Modifica parcialmente un recurso en el servidor.


Para enviar un formulario HTML es necesario utilizar el método POST.

Para acceder a una página a través de un navegador, es necesario utilizar el método GET, junto con la URL del recurso que se desea mostrar.

El código 200 indica que la petición realizada al servidor se ha llevado a cabo con éxito.

El código 404 indica que el recurso requerido no está disponible, por lo que no suele ser responsabilidad del desarrollador corregirlo

El código 500 se manda cuando el servidor encuentra una condición inesperada y no puede satisfacer la petición del usuario. Es responsabilidad del desarrollador asegurar que el servidor funcione correctamente.

Un elemento depreciado es aquél que ya no forma parte de la especificación del lenguaje. Algunos navegadores mantienen el soporte para componentes obsoletos para asegurar la compatibilidad con una mayor cantidad de páginas.

Algunos elementos depreciados:
  • applet - Inserta una applet de java. Actualmente se usa object.
  • center - Centra los elementos dentro de la etiqueta. Ahora se hace desde la hoja de estilos o utilizando align="center"
  • font - Aplica una fuente. Ahora se hace en CSS.
  • menu - Para listas de menú. Actualmente se utiliza li.
